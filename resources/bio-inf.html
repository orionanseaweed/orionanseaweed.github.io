Bioinformatics and Computational Biology

Computational Biology

  • A Primer for Computational Biology by Shawn T. O'Neil.

Bioinformatics

  • Essentials Bioinformatics (2006) by Jin Xiong.
  • Bioinformatics and Functional Genomics (2015) by Jonathan Pevsner.

Introduction to R (link to Data Science list)

  • An introductory course showing some basics and also some powerful things R can do (things that would be more difficult to do with Excel): Introduction to R for Biologists.
  • A book contains practical and theoretical aspects of computational genomics, helps readers analyze their own genomics data: Computational Genomics with R.
  • R packages: gggenomes, gggenes.

Introduction to Python

  • Python for Data Analysis (3e) by Wes McKinney.

Online Courses & Classes

  • An advanced course from HarvardX: HarvardX Biomedical Data Science Open Online Training - an impressive MOOCs I would say.
    Including: Data Analysis for the Life Sciences, Genomics Data Analysis and Using Python for Research.
  • An intermediate course on essential statistics, R and Python programming, offering challenges through homeworks, labs and projects: SDS 348 Spring 2020, The Wilke Lab.
    Lecture recommendations (with Python): Running queries on Entrez, regular expression, aligning sequences, BLAST, phylogenetic trees, geospatial data.
  • An intermediate Coursera course with certification focusing on visualization: Data Visualization for Genome Biology.
    You will learn to use:
    1. Galaxy: an open source, web-based platform for data intensive biomedical research.
    2. JBrowse: a modular genome browser with views of synteny and structural variation.
    3. Cytoscape: a software environment for integrated models of biomolecular interaction networks.
    4. Interactive Tree of Life: create, manage, annotate, and export high quality tree figures for publications.
