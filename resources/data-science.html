Data Science List

My list primarily focuses on data science with R - a powerful programming language for statistical computing and data visualization.

Textbooks

Data Science

  • R for Data Science (2e).
  • ggplot2: Elegant Graphics for Data Analysis (3e).

Data Visualization

  • Data Visualization: A practical introduction.
  • Fundamentals of Data Visualization.

R Documents Focusing on Visualization

  • R Graphics Cookbook (2e).
  • Modern Data Visualization with R.

Advanced R

  • A book designed for R users who want to improve their programming skills and understanding of the language: Advanced R (2e).
  • A book contains practical and theoretical aspects of computational genomics, helps readers analyze their own genomics data: Computational Genomics with R.
  • A book aims at data scientists who want to get a grip on using spatial data in their analysis: Spatial Data Science With Applications in R.

Additional Resources

Introduction to Related Tools & Applications

  • RStudio Cheatsheets.
  • R Markdown: The Definitive Guide.
  • Plotly R Open Source Graphing Library.
  • A professional non-coding vector graphics software for refining plots: Inkscape and A Guide to Inkscape.
  • You can build professional interactive web apps using the shiny package (Mastering Shiny & Shiny for R Gallery) or more simply with flexdashboard via R Markdown.

Online Courses & Classes

  • An introductory course showing some basics and also some powerful things R can do (things that would be more difficult to do with Excel): Introduction to R for Biologists.
  • An intermediate course on essential statistics, R and Python programming, offering challenges through homeworks, labs and projects: SDS 348 Spring 2020, The Wilke Lab.
  • Data Visualization & Dashboarding with R Specialization.
  • Data Visualization for Genome Biology.
